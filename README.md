# 4-bit & 8-bit Calculator (DLD Project)

A Digital Logic Design project implementing a calculator using fundamental logic circuits.
The project includes both a **physical 4-bit calculator** and **Proteus simulations for 4-bit and 8-bit calculators**.

---

## Objective

To design and implement arithmetic operations using digital logic components and understand low-level computation through hardware and simulation.

---

## Project Components

### Physical Implementation

* 4-bit calculator built using logic gates and ICs
* Circuit wiring and real hardware testing

---

###  Proteus Simulation

* 4-bit calculator simulation
* 8-bit calculator simulation
* Designed and tested using Proteus

---

## Features

* Basic arithmetic operations (e.g., addition, subtraction,multiplication)
* Binary input/output handling
* Scalable design from 4-bit to 8-bit

---

## Tools & Technologies

* Proteus Design Suite
* Digital Logic Components (ICs, gates, etc.)

---

##  Project Structure

* `proteus/` → Simulation files (.pdsprj,)
* `hardware/` → Physical circuit report

---

## How to Use

### Proteus Simulation

1. Open `.pdsprj` file in Proteus
2. Run the simulation

### Physical Circuit

* Refer to circuit images for wiring
* Provide inputs and observe outputs
